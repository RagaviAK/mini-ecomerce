This project is a fully functional Mini Ecommerce Web Application built using the MERN stack. It includes product listing, product detail pages, cart management, order placement, and search functionality. The application features a clean user interface, smooth navigation, and a scalable backend architecture.

---

## Features

### Home Page
- Displays all available products with a clean and responsive layout  
- Supports mobile, tablet, and desktop screen sizes  
- Includes a search bar for instant product filtering  

### Product Details Page
- Shows product image, description, category, and price  
- Implements dynamic routing for each product  
- Provides easy navigation back to the home or cart page  

### Cart System
- Allows adding items to the cart from the product page  
- Displays item details, quantity, and pricing  
- Supports item removal and quantity updates  
- Cart updates persist until order placement  

### Order Placement
- Includes a checkout page to confirm the order  
- Orders are stored in the MongoDB database  
- Backend handles validation and order creation logic  

### Backend
- Developed using Node.js and Express.js  
- Organized REST API structure  
- MongoDB used for storing products, cart items, and orders  
- Modular route and controller architecture
